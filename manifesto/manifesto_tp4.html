Resumo:

Este trabalho foi sobre a alteração de um ficheiro RDF para incluir uma nova classe: Moléculas. Esta classe tem informação sobre os elementos constituintes da molécula, o nome da molécula e, caso exista, o ponto de evaporação e ponto de fusão da molécula.

Para realizar esta alteração, importei os dados do ficheiro original RDF para a ferramenta Protégé, onde foi adicionada a nova classe assim como as novas propriedades.

Finalmente, foram criados indivíduos pertencentes à nova classe e usando as novas propriedades. Estas moléculas foram: água (H2O), dióxido de carbono (CO2), amoníaco (NH3), metano (C2H6), glicose (C6H12O6) e ácido súlfúrico (H2SO4)
